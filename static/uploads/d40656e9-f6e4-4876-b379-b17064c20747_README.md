# VirtualRoom - Hệ thống học tập trực tuyến

## 🎯 **Mô tả**
Hệ thống học tập trực tuyến với tính năng tạo câu hỏi tự động từ video bài học, sử dụng AI Gemini và diversity manager để tránh lặp câu hỏi.

## 🚀 **Tính năng chính**
- **Tạo phụ đề tự động** từ video bài học
- **Tạo câu hỏi tự động** từ nội dung video (MCQ, Essay, Oral)
- **AI Gemini integration** - Tạo câu hỏi từ video content thực tế
- **Diversity management** - Tránh lặp câu hỏi
- **Quiz system** - Hệ thống kiểm tra đa dạng

## 📋 **API Endpoints**

### **Quiz API**
- `POST /api/ai-quiz/question` - Tạo câu hỏi từ video
- `POST /api/ai-quiz/batch-questions` - Tạo nhiều câu hỏi
- `GET /api/ai-quiz/video-analysis` - Phân tích video

### **Caption API**
- `POST /api/captions/auto` - Tạo phụ đề tự động

### **Course API**
- `GET /courses` - Danh sách khóa học
- `POST /courses` - Tạo khóa học mới
- `GET /courses/<id>` - Chi tiết khóa học

## 🚀 **Sử dụng**

### **1. Tạo phụ đề tự động**
```javascript
// Trong course_detail.html
btnGen.addEventListener('click', async () => {
  const resp = await fetch('/api/captions/auto', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ 
      video_url: videoUrl, 
      course_id: courseId 
    })
  });
});
```

### **2. Tạo câu hỏi từ video**
```javascript
// Tạo câu hỏi MCQ
const resp = await fetch('/api/ai-quiz/question', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    course_id: '123',
    type: 'mcq',
    caption_url: 'video.vtt'
  })
});
```
