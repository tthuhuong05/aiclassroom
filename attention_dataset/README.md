# Mini Attention Dataset (Demo)

This folder contains a tiny, hand-labeled set for attention/pose:
- `attention_label`: `attentive` or `not_attentive`
- `pose_label`: one of `look_down`, `look_up`, `look_left`, `look_right`, `forward`
- `cheat_label`: `yes`/`no` (whether a cheat tool is visible).

These sample images are all labeled as `not_attentive` because the student looks away from the screen.
Use this as a template to expand with your own frames.
